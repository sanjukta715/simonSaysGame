# Simon Says Game 

A simple browser-based Simon Says memory game built with **HTML, CSS, and JavaScript**. Test your memory by following the color sequence and see how far you can go!

## How to Play
- Click the buttons in the same order as the sequence shown.
- Repeat the sequence correctly to advance levels.
- Game ends if a wrong button is clicked.

## Technologies
HTML | CSS | JavaScript
